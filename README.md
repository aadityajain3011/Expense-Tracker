# Expense Tracker

## Definition
This project is a simple Expense Tracker Dashboard built using HTML and CSS. It displays monthly expenses in a clean and organized format.

## Problem Statement
Create a web-based dashboard that shows expense details such as total spending, highest expense category, and recent transactions using only HTML and CSS.

## Features
- Dashboard layout with header section
- Summary cards (Total Spend, Highest Category, Current Month)
- Expense categories with amount display
- Recent transactions list
- Clean and responsive UI using Flexbox

## Technologies Used
- HTML
- CSS (Flexbox)

## Output
The dashboard displays:
- Total monthly spending
- Category-wise expenses
- Recent transactions with date and amount
